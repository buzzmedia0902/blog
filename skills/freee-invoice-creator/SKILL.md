# freee請求書作成スキル

## 概要

このスキルは、freee会計システムを使用してAibee研修カリキュラムの請求書を作成するための標準ワークフローを提供する。助成金管理システムとの連携により、顧客情報を効率的に参照しながら、正確な請求書を作成できる。

## ワークフロー

### 1. freeeへのログインとナビゲーション

freee会計システムにログインし、請求書作成画面に移動する。

**手順**:
1. freeeログイン画面を開く: `https://accounts.secure.freee.co.jp/login`
2. 既存のアカウント（accounting@buzzmedia.co.jp）でログインする
3. 「請求入金」ボタンにカーソルを合わせる（クリックではなくホバー）
4. 表示されるトグルメニューから「請求書」を選択
5. 右上の青い「プラス新規作成」ボタンをクリック

### 2. 取引先情報の登録または参照

新規顧客の場合と既存顧客の場合で手順が異なる。

#### 新規顧客の場合

新規登録フォームに必要最小限の情報を入力する。

**手順**:
1. 「新規登録」を選択
2. 指示された会社名を入力
3. メールアドレスが判明している場合は入力
4. その他の項目は情報がない場合は空欄でOK
5. 口座情報は入力不要
6. 青い「登録」ボタンをクリック

#### 既存顧客の場合

助成金管理システムから顧客情報を参照する。

**手順**:
1. MCPツール `search_customers` で顧客を検索
2. MCPツール `get_customer` で顧客の詳細情報を取得
3. MCPツール `get_application_data` で申請フォームデータを取得
4. 取得した情報を確認:
   - 会社名
   - 代表者名
   - 会社住所
   - その他の連絡先情報
5. freeeの取引先入力欄に会社名を入力し、既存の取引先を選択

**注意**: 助成金管理システムから取得した代表者名と、freeeに登録されている担当者名が異なる場合がある。スキルの固定値（代表取締役 今村雄飛）は自社担当者であり、顧客の担当者名ではない。

### 3. 請求書基本情報の入力

請求書の基本情報を設定する。固定値と可変値を正確に区別すること。

#### 入力順序の重要性

freeeのフォームは自動補完機能があるため、以下の順序で入力することを推奨:

1. 取引先を選択
2. **ページを上にスクロール**して入金期日、件名、自社担当者の入力欄を表示
3. 入金期日を入力
4. 件名を入力
5. 自社担当者を入力
6. **ページを下にスクロール**して明細入力欄を表示
7. 明細を入力

#### 固定項目

以下の項目は常に同じ値を使用する:
- **自社担当者**: 代表取締役 今村雄飛
- **件名**: Aibee実務活用AIカリキュラム利用料

**重要**: 「宛先」は顧客の担当者名であり、助成金管理システムから取得した代表者名を使用する。「自社担当者」は自社側の担当者名であり、常に「代表取締役 今村雄飛」を使用する。

#### 可変項目

以下の項目は状況に応じて設定する:

**請求日**:
- 基本的に当日の日付を使用
- 指示がある場合は指定された日付を使用

**入金期日**:
- 当月の翌月末を設定（例: 2月に作成する場合は3月31日）
- 土日の場合は、その前の営業日に調整
- 計算例:
  - 2026年2月19日に作成 → 入金期日: 2026年3月31日
  - 3月31日が土曜日の場合 → 3月29日（金曜日）に調整

### 4. 請求書明細の入力

請求書の明細行に品目、数量、単価を入力する。

**手順**:
1. 「明細」セクションで以下を入力:
   - **摘要**: Aibee実務活用AIカリキュラム（固定）
   - **数量**: 研修規模に記載された受講人数を入力（下記の「数量欄の入力方法」を参照）
   - **単位**: 名
   - **単価**: 400000（40万円）

2. 「テキスト行を追加」をクリック
3. テキスト行に研修日程を入力（指示された日程を記載）

#### 数量欄の入力方法（重要）

freeeのフォームは自動補完機能が干渉するため、数量欄の入力には特別な手順が必要:

1. 数量欄をクリックしてフォーカス
2. Ctrl+Aで全選択
3. Deleteキーを押す（これにより正しい値が入力される）
4. 単位欄など別の欄をクリックしてフォーカスを外す
5. 金額が正しく再計算されたことを確認

**注意**: `browser_input`ツールで直接入力すると、既存の値に追加されてしまい「15」や「155」になる。必ず上記の手順で入力すること。

### 5. 税区分の設定

単価40万円は税込み価格のため、必ず「内税」に設定すること。

#### 税区分の変更方法（重要）

明細行の税区分ドロップダウンは自動化ツールで操作できないため、以下の手順で変更する:

1. 「課税・表示設定」ボタンをクリック
2. ダイアログ内の「消費税」セクションで「税込表示（内税）」ラジオボタンをクリック
3. 「保存」ボタンをクリックして設定を適用

これにより、請求書全体の税区分が内税に変更され、合計金額が再計算される。

**税区分の重要性**:
- 単価40万円は税込み価格のため、必ず「内税」に設定すること
- 「外税」のままだと、40万円に消費税が加算されてしまう（合計2,200,000円になる）
- 「内税」に設定すると、40万円が税込み価格として扱われる（合計2,000,000円になる）

**設定内容**:
- **源泉徴収**: なし（チェックを入れない）
- **消費税**: 内税で設定

**注意**: 源泉徴収は仕事を依頼した時にかかる税金のため、本ケースでは適用しない。

### 6. 保存とPDFダウンロード

請求書を保存し、PDFファイルとしてダウンロードする。

**手順**:
1. 入力内容を確認
2. 画面下部の青い「保存」ボタンをクリック
3. 保存完了後、右側のプレビュー画面上部にある「ダウンロード」ボタンをクリック
4. 「PDFダウンロード」を選択
5. ダウンロードされたPDFファイルを確認

**注意**: PDFダウンロードボタンは、請求書詳細ページの右側プレビュー画面上部にある。

## トラブルシューティング

### 数量が正しく入力できない

**症状**: 数量欄に「5」を入力しようとすると「15」や「155」になる

**原因**: freeeのフォームの自動補完機能が干渉している

**解決方法**: 上記「数量欄の入力方法」の手順に従う

### 税区分のドロップダウンが操作できない

**症状**: 明細行の税区分ドロップダウンをクリックしても選択肢が表示されない

**原因**: 自動化ツールでは操作できない仕様

**解決方法**: 上記「税区分の変更方法」の手順に従い、「課税・表示設定」から変更する

### 件名や自社担当者が重複して入力される

**症状**: 件名や自社担当者の入力時に、既存のテキストに追加されてしまう

**原因**: フォームの自動補完機能が干渉している

**解決方法**: 
1. ページを上にスクロールして入力欄を表示
2. 入力欄をクリックしてフォーカス
3. Ctrl+Aで全選択
4. 新しい値を入力

## 重要な注意事項

### 税区分の設定
単価が税込み40万円であるため、必ず「内税」に設定すること。これを忘れると請求金額が誤って計算される（2,200,000円になる）。

### 入金期日の計算
翌月末が土日の場合は、必ず前営業日に調整すること。祝日の考慮が必要な場合は、カレンダーを確認する。

### 顧客情報の参照
助成金管理システムに顧客情報がある場合は、必ず参照して正確な情報を転記すること。手入力によるミスを防ぐため。

### 固定値の遵守
自社担当者(代表取締役 今村雄飛)と件名(Aibee実務活用AIカリキュラム利用料)は常に固定値を使用すること。

### 宛先と自社担当者の区別
- **宛先**: 顧客の担当者名（助成金管理システムから取得）
- **自社担当者**: 自社側の担当者名（常に「代表取締役 今村雄飛」）
